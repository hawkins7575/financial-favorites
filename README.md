# 프로젝트 개발 문서: 금융 즐겨찾기 (Financial Bookmarks Portal)

## 1. 프로젝트 개요 (Overview)
**프로젝트명**: 금융 즐겨찾기 (Financial Hub)
**목적**: 인터넷에 산재해 있는 주식, 코인, 채권, 보험 등 다양한 금융 관련 웹사이트 및 유튜브 채널을 카테고리별로 모아 한눈에 쉽게 확인하고 이동할 수 있는 직관적인 포털 서비스입니다.
**핵심 목표**: "복잡한 금융 지식을 탐색하기 위한 가장 직관적이고 편안한 첫 화면 제공"

---

## 2. 주요 기능 명세 (Core Features)

### 2.1 사이트 탐색 및 필터링
* **다중 필터링 시스템**: 
  * **자산 카테고리 (Category)**: 전체, 주식, 코인, 채권, 보험, 일반금융 등 주제에 맞춰 링크를 분류.
  * **콘텐츠 형식 (Type)**: 웹사이트, 유튜브 비디오/채널 등 미디어 성격에 따른 분류.
* **직관적인 상태 표시**: 현재 선택된 필터 버튼을 밝고 명확하게 하이라이트(Active 상태) 하여 현재 탐색 중인 영역을 사용자에게 직관적으로 알려줍니다.

### 2.2 사이트 카드 표시 (Grid Layout)
* **카드 뷰 아키텍처**: 각 웹사이트를 하나의 카드로 시각화하여 그리드 형태로 배치.
* **카드 구성 요소**: 
  * 식별을 돕는 아이콘 (FontAwesome 활용)
  * 사이트 제목
  * 간단한 요약 설명
  * 바로가기 링크 버튼 

### 2.3 사용자 정의 데이터 관리
* **북마크 추가 기능 (Modal)**: 
  * 화면 상단의 "+ 추가" 버튼을 통해 새로운 링크를 등록.
  * 입력 항목: 제목, URL, 카테고리, 형식, 설명, 아이콘 클래스.

## 3. UI/UX 디자인 가이드라인 (Design System)

금융 서비스 특유의 '신뢰성'과 현대적인 '프리미엄 핀테크(FinTech)'의 세련됨을 결합합니다.

* **색상 팔레트 (Color Scheme)**
  * **배경**: 심도 있는 다크 모드 (완전한 블랙보다는 다크 네이비 혹은 차콜 그레이 사용).
  * **포인트 컬러**: 형광빛이 도는 블루(`Neon Blue`)나 그린(`Mint Green`)을 사용하여 액티브 요소와 버튼을 강조. 밝고 대비가 강해 버튼의 시인성이 높습니다.
  * **텍스트 영역**: 가독성을 위해 차분한 화이트/오프화이트 및 회색 계열 활용.

* **타이포그래피 (Typography)**
  * 폰트: **Pretendard** (현대적이고 가독성이 매우 뛰어난 고딕 폰트 렌더링).

* **인터랙션 및 애니메이션 (Micro-interactions)**
  * 호버 효과: 카드나 버튼에 마우스를 올렸을 때 부드럽게 떠오르거나(Transform: translateY), 빛나는 효과(Glow / Box-shadow)를 추가해 생동감을 줍니다.
  * 모달 및 필터 트랜지션: 화면이 전환될 때 페이드인(Fade-in) 되어 사용자가 시각적으로 편안함을 느끼도록 합니다.

## 4. 시스템 아키텍처 및 기술 스택 (Tech Stack)

이 프로젝트는 현재 클라이언트 사이드 기술로 매우 가볍게 구동되도록 설계되었습니다.

1. **HTML5**: 시맨틱 태그 구조 및 전반적인 페이지 뼈대 (index.html).
2. **CSS3 (Vanilla)**: Grid 및 Flexbox 레이아웃, 모바일 반응형(Media Query) 미디어 뷰. 사용자 지정 속성(CSS Variables)을 적극 활용한 일관된 토큰 관리 (style.css).
3. **JavaScript (Vanilla JS)**: 상태 관리, DOM 요소 동적 생성, 필터링 로직 구현 (script.js).
4. **Data 파일 분리**: 초기 사이트 목록을 별도의 스크립트로 분리하여 관리를 용이하게 구성 (data.js).

## 5. 향후 확장 및 고도화 계획 (Future Work)

1. **로컬 데이터 영구 저장 (Local Storage 연동)**: 현재는 메모리 상에서만 추가되지만, Web Storage API를 활용해 새로고침 후에도 추가한 즐겨찾기 유지.
2. **검색(Search) 기능 강화**: 필터 상단에 검색 바를 탑재하여 키워드로 즉각적인 사이트 탐색 기능 도입.
3. **드래그 앤 드롭 (Drag & Drop)**: 카드들의 배치 순서를 사용자가 원하는 대로 커스터마이징 기능 추가.
4. **테마 전환**: 사용자 기호에 맞춘 라이트/다크 모드 스위치 구축.
